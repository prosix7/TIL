### CSS 박스 모델
---
#### border-width 속성
<p>테두리의 두께를 지정한다. 2개 지정 시, top과 bottom을 첫 번째 속성 값으로, rigth과 left를 두 번째 속성 값으로 적용한다.</p>
<p>4가지 속성값 지정시 top > right > bottom > left 순으로 적용한다.</p>

```html
#box1 { border-width: thin | medium | thick | px; }
```

#### border-color 속성
<p>테두리의 색상을 지정한다.</p>

```html
#box1 { border-color: red; }
```
