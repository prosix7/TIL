## Tags
### 표 만들기
* table 태그로 표를 만들고 caption 태그로 제목을 붙인다.
* tr 태그는 행을, td와 th 태그는 셀을 만든다.
```html
<table>
  <caption>제목</caption>
  <tr>
    <td>1행 1열</td>
    <td>1행 2열</td>
    <td>1행 3열</td>
  </tr>
  <tr>
    <td>2행</td>
  </tr>
</table>
```
이상하게 예제대로 하면 이쁜 선이 안나오는데 이때 style 태그를 이용하자
```html
<style>
        table, td, th {
            border: 1px solid black;
            border-collapse: collapse;
        }
    </style>
```
